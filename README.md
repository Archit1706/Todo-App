## To Do App

Complete Todo App with CRUD Operations

### Project Structure

1. React JS - Frontend
2. Express, Node Js and Mongo DB - Backend

### Steps to Run

1. cd into frontend folder
2. `npm install`
3. `npm start`
4. cd into backend folder (rename the .env.example file to .env)
5. `npm install`
6. `node index.js`
